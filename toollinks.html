The following tools and libraries may greatly help in cross-platform projects based on C++ (verified):

 CImg library
  Graphic image creation, manipulation, drawing, and displaying in a window.
  Built-in BMP file save/load.
  The simplest possible structure of sources: 1 header file only, with 4 classes.
  May be compiled and work stably on all major desktop platforms.
cimg.eu   CImg_latest.zip

 PortAudio library
  C wrapper for playing and recording audio.
  Provides simple callbacks for separate and joint audio samples I/O, with explicit latency control.
  May be compiled and work stably on all major desktop platforms.
  Supports several system audio APIs per platform, selectable at run time.
portaudio.com

 QtCreator
  IDE around multi-purpose Qt libraries, supporting large number of platforms.
  Flexible code editor, with maybe the lowest (!) UI response time on C++ code navigation and completion.
  Able to handle and build projects without dependencies on Qt and other non-system libraries.
github.com/qt-creator   www.qt.io/download   wiki.qt.io/Qt_Creator
