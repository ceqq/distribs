BMDX library

(binary modules data exchange) 
Efficient passing high-level objects and data containers between executable and shared libraries (.dll, .so).
6 C++ source files only (5 headers + 1 transl. unit).
Tolerance to compiler differences (structures size, alignment, implementation, symbols binding).

Features (brief):

cross-module polymorphic wrapper for scalar, array, hash, map, strings;
object referencing and access through attached interfaces;
locks, thread control;
message sending (currently in-process only);
conversion between data trees and editable text;
.dll/.so loader;
file i/o, string utilities, process launcher etc.

All containers are heavily optimized.

Nearest planned features: interprocess and network message sending; alt. stack for clean thread termination.

Language compatibility: C++98/C99, C++11, C++14.
Dependencies: ONLY standard C, C++ libraries and platform headers.
Tested: all major desktop and mobile platforms, >20 compilers (released between 2008..2017).
