BMDX library

(binary modules data exchange) 
Efficient passing high-level objects and data containers between executable and shared libraries (.dll, .so).
6 C++ source files only (5 headers + 1 transl. unit).
Tolerance to compiler differences (structures size, alignment, implementation, symbols binding).

Features (brief):

cross-module polymorphic wrapper for scalar, array, hashlist, map, strings, objects of any type;
dynamically attaching interfaces to wrapped objects;
memory-efficient zero-copy vector and hash;
locks, thread control, simple multithread control for parallel computation;
in- and inter-process messaging, shared memory wrapper;
conversion between data trees and editable text;
unified .dll/.so loader;
unified utilities for file i/o, strings, dates, process launching etc.

All containers are heavily optimized.

Near plans: message dispatcher improvement.

Language compatibility: C++98/C99, C++11, C++14.
Dependencies: ONLY standard C, C++ libraries and platform headers.
Tested: all major desktop and mobile platforms, >20 compilers (released between 2008..2018).
